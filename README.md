## Project for studies of design patterns

![Java Build](https://github.com/lucasantonioo/design-pattern-labs/actions/workflows/maven.yml/badge.svg?branch=master)

The objective of this project is to develop good strategies according to each use case, applying the design patterns in the simplest possible way.

As the focus here is design pattern, there is no need to implement complex functions or use frameworks, especially database connection.

The implementation of the developed strategies is exclusively through unit tests.

## Technologies

- Java 17
- JUnit 5.7.0
- Maven
